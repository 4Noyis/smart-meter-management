# Smart-Meter-Management-System


Create environment 
```
    cd project_folder
    python3 -m venv .venv
```

Activate environment
```
    . .venv/bin/activate
```

Install Flask

```
    pip install Flask
```

Run app and go to the given http port on your browser
```
    flask --app app run
```
